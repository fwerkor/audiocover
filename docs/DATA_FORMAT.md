# Data format

## Training input

A folder containing audio files:

- `.wav`
- `.flac`
- `.mp3`
- `.m4a`
- `.aac`
- `.ogg`

Recommended:

- 48 kHz WAV
- dry voice
- no effects
- no background music
- no clipping
- stable microphone position

## Model package

Generated by:

```bash
audiocover train raw_data models/my_profile --consent
```

Format:

```text
models/my_profile/
  model.yaml
  training_report.json
  dataset/
    report.json
    manifest.jsonl
    wavs/
  simple_timbre.json   # if backend=simple-timbre
  model.pth            # if external backend created it
  model.index          # if external backend created it
```

## Render input

One song file accepted by FFmpeg. Output is always WAV.
